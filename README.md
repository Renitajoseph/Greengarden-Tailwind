# 🌿 GreenGarden

A clean and responsive plant shop website built with **Tailwind CSS**, featuring Home, Products, and Contact pages. Ideal for showcasing interior plants with beautiful imagery and easy user experience.

## ✔️ Features

- **Responsive Navigation** for seamless switching between pages  
- **Hero Section** with call-to-action buttons  
- **About / Services Section** including features like “Large Assortment”, “Free & Fast Shipping”, and “24/7 Support”  
- **Best Seller Products** grid with product cards and 'Buy Now' buttons  
- **Customer Reviews** section styled with Tailwind  
- **Contact Form** for user inquiries  
- **Consistent footer** across all pages

🎨 Customization
Tailwind CSS via CDN (v2.2.19) — change to a later version via the <link> tag if needed

Styling — use Tailwind utility classes; you can also swap to a custom build or upgrade to Tailwind v3+

Add images — place new plant images in the /assets/ folder and update products.html accordingly

Expand site — create additional pages (e.g. Blog, FAQ) and link them via the navbar
